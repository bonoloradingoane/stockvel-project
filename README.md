# stockvel-project
A decentralized Stockvel (community savings group) built during the University of Johannesburg Blockchain Week.
This project explores how blockchain can improve transparency, trust, and automation in community financial systems.

# Features
Smart contract for group contributions
Automated payout cycles
Member registration & tracking
Transparent contribution history
Simple front-end interaction (if included)

# Tech Stack
Solidity (smart contracts)
Ethereum / EVM-compatible testnet
Remix IDE / Hardhat (depending on your version)
MetaMask for contract interaction

# Learning Outcomes
Understanding decentralized savings models
Building and deploying smart contracts
Handling on-chain logic for group finance
Secure membership + payout flows

# Status
This is a learning project created during UJ Blockchain Week.
More improvements will be added as I continue building.
